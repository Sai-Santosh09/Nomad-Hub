# NomadHub

NomadHub is a compact travel-inspired USB hub PCB designed in **KiCad**. It provides reliable USB connectivity through one USB Type-C upstream port, two USB Type-C downstream ports, and two USB Type-A downstream ports using the SL2.1S USB 2.0 Hub Controller.

## Features

- Compact travel-friendly PCB
- USB Type-C upstream port
- 2 × USB Type-C downstream ports
- 2 × USB Type-A downstream ports
- SL2.1S USB 2.0 Hub Controller
- 2-layer PCB design
- USB 2.0 compliant routing
- Gerber files included
